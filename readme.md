### Run the Project

1. Clone and enter the repo:
   ```bash
   git clone https://github.com/JaZax/swift-codes-api
   cd 
   ```

2. Run the docker container [docker (hyperlink)]:
    ```bash
    docker-compose up
    ```

3. API: Access at http://localhost:8080
    Database: 
    ```bash
    mongosh "localhost:27017"
    ```

4. api documentation:
    http://localhost:8080/docs
